Instale o XAMPP

Após instalar, vá até o disco C na pasta xampp, entre em htdocs e crie uma nova pasta com o nome do projeto (Ou copie a pasta baixada do drive direto para lá).

Compie os arquivos do projeto para essa nova pasta.

Inicie o servidor apache e o mysql no aplicativo xampp

No seu navegador, digite localhost/nome_da_pasta_do_projeto
